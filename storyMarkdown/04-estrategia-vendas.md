## Framework Atual vs. Recomendado

### Atual: Fluxo LINEAR Rígido (10 etapas sequenciais)
```
Qualificação → Catálogo → Modelo → Logo → Posição → Extras →
Mockup → CEP → Orçamento → Pagamento → Dados → Pedido
```
**Problema:** Zero flexibilidade. Cliente DEVE seguir todas as etapas na ordem.

### Recomendado: Fluxo ADAPTATIVO com Atalhos
```
Qualificação → Preview Rápido (3 opções in-chat)
                    ↓
              Escolha → Personalização Agrupada
                    ↓
              Mockup + Preço Estimado (juntos)
                    ↓
              Aprovação → CEP + Frete
                    ↓
              Orçamento Final → Pagamento
                    ↓
              Dados Envio → Pedido
```

### Diferenças-chave:
1. **Preview in-chat** em vez de link externo
2. **Mockup + preço estimado juntos** (reduz ansiedade de preço)
3. **Personalização agrupada** (1 mensagem em vez de 6)
4. **Dados de envio ANTES do pagamento** (resolve objeção "nem tem meu endereço")

## Aplicação dos Frameworks de Vendas

### SPIN Selling (Adaptado para Chat)

**S**ituação: Já vem do SDR com segmento, quantidade, modelo
**P**roblema: Não explorado atualmente. Oportunidade:
```
"Sua equipe trabalha muito no sol, né? Proteção UV faz toda
diferença no dia a dia!"
```
**I**mplicação: Não usado. Poderia ser:
```
"Imagina sua equipe toda uniformizada, passando profissionalismo
pros clientes!"
```
**N**eed-Payoff: Parcialmente usado no mockup. Reforçar:
```
"Olha como ficou a cara do seu time! 🚀 [mockup]"
```

**Recomendação:** Inserir 1 frase de SPIN entre a qualificação e o catálogo.

---

### PAS (Problem-Agitate-Solution) - Para Abertura

**P**roblem:
```
"Trabalhar no sol sem proteção adequada é complicado, né?"
```
**A**gitate:
```
"Camisas comuns não protegem de verdade e ainda desbotam rápido."
```
**S**olution:
```
"Nossas camisas UV50+ são feitas pra aguentar o tranco!
E ainda ficam com a cara do seu time 😊"
```

**Quando usar:** Na mensagem de abertura, ANTES de enviar catálogo.
**Nota:** Para oversized, malha-algodão e polo, adaptar sem mencionar UV.

---

### AIDA (Attention-Interest-Desire-Action) - Para Mockup

**A**ttenção: O disclaimer atual ("⚠️ Esse mockup é apenas ilustrativo") MATA a atenção. Deveria vir DEPOIS.

**I**nteresse: Mostrar o mockup com entusiasmo ANTES do disclaimer.

**D**esejo: Comparar com camisas genéricas. "Olha a diferença!"

**A**ção: CTA claro e único. "Aprovado? Vou montar seu orçamento!"

**Atual (ordem errada):**
```
⚠️ Esse mockup é apenas ilustrativo. [disclaimer]
Olha só como ficou: [mockup]
O que achou? Aprova esse modelo? 😊
```

**Recomendado (AIDA):**
```
Olha como ficou a camisa do seu time! 🤩 [mockup]

Aprovou? Se sim, já monto o orçamento com frete pra você!

📌 Lembrando que esse é só o preview — nosso designer
faz a versão final e te envia pra aprovação antes de produzir 😊
```

## Estratégia de Preço

### Problema Atual
O preço é revelado APENAS na etapa 5 (de 10), após ~20-40 minutos de conversa. Clientes sensíveis a preço abandonam antes de chegar lá.

### Estratégia Recomendada: Ancoragem Progressiva

**Etapa 1 (Abertura):** Faixa geral
```
"Nossas camisas personalizadas vão de R$49,90 a R$119,90"
```

**Etapa 3 (Após escolha de modelo):** Preço específico
```
"Manga Longa UV50+ sai R$79,90 cada — qualidade premium!"
```

**Etapa 5 (Orçamento):** Total com frete
```
"3x Manga Longa UV50+ = R$239,70
+ Frete PAC = R$22,50
= TOTAL: R$262,20 (ou 12x de R$21,85)"
```

### Benefícios:
- Cliente já sabe a faixa ANTES de investir tempo
- Cada revelação de preço é MENOR que a expectativa anterior
- O parcelamento é apresentado junto com o total (reduz choque)

| Objeção | Frequência | Resolução Atual | Eficácia | Recomendação |

### Nova Estratégia por Objeção:

**"Tá caro" → Técnica do Custo por Uso:**
```

sai R$6,66 por mês. Menos que um café por semana pra ter seu
time protegido e uniformizado! 😊

E dá pra parcelar em até 12x. Quer que eu monte o orçamento
completo?
```

**"Quero humano" → Escalação Inteligente (após 2ª tentativa):**
```
[1ª vez] "Posso te ajudar com tudo aqui! Sou especialista em
personalização 😊 Em que posso te ajudar?"

[2ª vez] "Entendo sua preferência! Vou registrar seu pedido e
nosso time entra em contato em até 2 horas. Pode ser? 😊"
→ Executar transferir_humano
```

**"Já tenho design" → Abraçar:**
```

pro mockup. Nosso designer vai fazer certinho igual!

Só preciso saber o modelo de camisa (manga longa, curta...)
e o tamanho da equipe!
```

## Estratégia de Follow-up

### Timing Otimizado (baseado em dados do mercado brasileiro)

| Tempo | Tipo | Exemplo |
|---|---|---|
| 30 min | Ajuda contextual | "Posso te ajudar a escolher? Qual estilo sua equipe prefere?" |
| 4 horas | Social proof + valor | "O time da [empresa] do seu segmento acabou de personalizar!" |
| 24 horas | Urgência real | "Produção fecha sexta — dá tempo de garantir pro seu time!" |
| 48 horas | Mockup recovery | Enviar mockup personalizado do segmento como isca |
| 7 dias | Re-engajamento | "Oi [Nome]! Ainda pensando nas camisas? Posso tirar alguma dúvida?" |

### Regras:
- **NUNCA** enviar follow-up 14 min após silêncio (muito cedo)
- **NUNCA** dizer "vou encerrar" — deixar aberto
- **SEMPRE** personalizar com dados já coletados
- **Máximo** 3 follow-ups por conversa

## Estratégia de Pagamento

### Problema: Coleta de CPF/Email Gera Abandono

Dos 3 leads que chegaram ao pagamento, nenhum completou.
Principal barreira: pedido de CPF + email.

### Solução: Simplificar Coleta

**Atual (4 campos de uma vez):**
```
Preciso de:
- Nome completo
- CPF
- E-mail
- Telefone
```

**Recomendado (progressivo):**
```
"Pra gerar seu pagamento, me confirma seu nome completo? 😊"
[resposta]
"E o CPF? (só pra emitir a nota fiscal)"
[resposta]
"E um email pra te enviar a confirmação?"
[resposta]
→ Gerar link
```

### Justificativa: Explicar POR QUE precisa de cada dado reduz desconfiança.
